Tentang dashboard ini
Dashboard Monitoring Survey General memantau seluruh aktivitas survei lapangan proyek PLTS yang bekerja sama dengan SUN Energy. Data mencakup surveyor, PIC Engineering, status progres, serta nominal invoice mandays dan reimbursement per proyek. Data diambil langsung dari Google Sheets via Apps Script.

Ekspor Data

Unduh data survei dalam berbagai format

Pilih format ekspor

Ekspor CSV
Seluruh data survei yang sedang difilter, dalam format CSV berenkode UTF-8 (kompatibel Excel).
Ekspor JSON
Seluruh data survei mentah dalam format JSON, cocok untuk integrasi sistem atau analitik lanjutan.
Cetak / PDF
Buka dialog cetak browser untuk menyimpan tampilan dashboard sebagai PDF atau mencetak langsung.
Salin ke Clipboard
Salin ringkasan data dalam format tab-separated, siap di-paste ke Google Sheets atau Excel.
